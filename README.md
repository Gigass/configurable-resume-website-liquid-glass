# 动态交互式个人简历网站模板

## 简介

这是一个基于 **Vue 3**, **Vite** 和 **TypeScript** 构建的高度可配置、动态交互的个人作品集与简历网站模板。您**无需编写任何代码**，只需通过修改一个配置文件，即可轻松生成属于您自己的、具有酷炫视觉效果的个人网站。

本项目旨在通过前沿的前端技术、流畅的动画效果和精致的视觉设计，帮助您全方位、沉浸式地展示您的专业技能、项目经验和个人优势。

网站深度融合了液态玻璃（Liquid Glass）美学、3D变换和微交互，致力于为访问者提供一个独一无二、印象深刻的浏览体验。

## 🚀 如何配置并搭建您自己的网站

### 第一步：准备您的个人信息

在开始之前，请准备好所有您想要在网站上展示的个人信息，包括：

*   **个人简介**: 您的姓名、头衔、个人总结等。
*   **核心优势**: 您的主要技能和专业特长。
*   **项目经验**: 您参与过的主要项目，包括项目描述、您的角色和使用的技术。
*   **开源贡献**: 您的开源项目列表。
*   **联系方式**: 您的电话、微信、邮箱等。
*   **个人图片**: 项目中用到的背景图片等素材。

### 第二步：修改核心数据文件

您网站的所有内容都由一个文件统一管理，这个文件是：

**`/public/data/sitedata.json`**

您只需要用您自己的信息替换掉这个文件里的内容即可。下面是该文件中每个部分的详细说明：

#### 1. `layout` - 网站布局

这部分控制网站的全局布局，主要是导航栏。

```json
"layout": {
  "navigation": [
    { "name": "首页", "to": "/" },
    { "name": "项目经历", "to": "/project" },
    { "name": "开源项目", "to": "/opensource" },
    { "name": "联系我", "to": "/contact" }
  ]
}
```
*   `name`: 导航链接上显示的文本。
*   `to`: 链接对应的页面路径。

#### 2. `home` - 首页内容

这部分控制着首页的所有信息。

```json
"home": {
  "hero": {
    "title": "您的姓名",
    "subtitle": "您的头衔或一句话介绍",
    "description": "一段详细的个人简介..."
  },
  "advantages": [
    {
      "icon": "👨‍💻",
      "title": "您的第一个优势",
      "summary": "优势的简短总结...",
      "details": ["详细描述1", "详细描述2"]
    }
  ]
}
```
*   `hero`: 首页首屏的个人信息。
*   `advantages`: 您的核心优势列表，网站会以卡片轮播的形式展示。

#### 3. `portfolio` - 项目作品集

这里存放您的项目经验。

```json
"portfolio": [
  {
    "company": "您所在的公司名称",
    "items": [
      {
        "title": "项目名称",
        "meta": "时间范围 | 您的角色",
        "size": "large",
        "description": ["项目详细描述..."],
        "tags": ["技术标签1", "技术标签2"]
      }
    ]
  }
]
```
*   这是一个列表，您可以按公司或类别组织您的项目。
*   `size`: 卡片大小，`large` 表示占用整行的大卡片，`normal` 为普通大小。

#### 4. `opensource` - 开源项目

这部分用于展示您的开源贡献。

```json
"opensource": {
  "intro": { ... },
  "projects": [
    {
      "name": "您的开源项目名",
      "repo": "您的GitHub用户名/仓库名",
      "url": "项目链接",
      "description": "项目描述..."
    }
  ],
  "githubStatsCard": { ... },
  "moreProjects": {
    "text": "想要探索更多我的项目吗？",
    "buttonText": "访问我的 GitHub 主页",
    "url": "您的GitHub主页链接"
  }
}
```
*   `projects`: 您的开源项目列表。`repo` 字段用于自动抓取 GitHub 仓库信息并生成统计卡片。
*   `githubStatsCard`: 用于配置 GitHub 统计卡片的样式。
*   `moreProjects`: 页面底部的链接，引导访问者查看您的 GitHub 主页。

#### 5. `contact` - 联系方式

这里配置您的联系信息。

```json
"contact": {
  "title": "联系我",
  "contacts": [
    { "label": "手机", "value": "您的手机号", "type": "tel" },
    { "label": "微信", "value": "您的微信号", "type": "text" },
    { "label": "邮箱", "value": "您的邮箱地址", "type": "mailto" }
  ],
  "social": {
    "buttonText": "发送邮件",
    "link": "mailto:您的邮箱地址"
  }
}
```
*   `type`: 定义了联系方式的类型，`tel` 会生成电话链接，`mailto` 会生成邮件链接。

### 第三步：替换个人图片

网站中使用的所有图片都存放在 `public/assets/` 目录下。您可以将自己的图片替换掉同名文件，以更新网站的视觉素材。

*   **轮播图背景**: `public/assets/lb/` 目录下的 `lb1.png` 到 `lb8.png`。
*   **其他图片**: 您可以根据需要在代码中修改图片路径。

### 第四步：在本地运行和部署

在您完成上述所有信息的修改后，就可以在本地运行您的网站了。

#### **环境要求**
- Node.js >= 18.x
- npm / pnpm / yarn

#### **安装依赖**
```bash
npm install
```

#### **开发模式**
运行以下命令启动本地开发服务器。服务通常在 `http://localhost:5173` 上可用。
```bash
npm run dev
```

#### **生产构建**
当您准备好部署网站时，运行此命令来构建项目。优化后的文件将输出到 `dist` 目录。
```bash
npm run build
```
之后，您只需将 `dist` 目录中的所有文件上传到您的服务器或静态网站托管平台即可。

## ✨ 核心亮点

- **数据驱动**: 整个网站的内容完全由 `sitedata.json` 文件驱动，实现了内容与视图的彻底分离。
- **沉浸式首页体验**: 液态玻璃美学、故事化卡片轮播、Ken Burns 特效等。
- **高级项目作品集展示**: 动态响应式网格、3D悬停交互、高性能入场动画。
- **GitHub API 集成**: 自动生成美观的 GitHub 仓库统计卡片。
- **统一且精致的视觉语言**: 以液态玻璃为核心设计语言，色彩与渐变运用得当，完美适配多端设备。
